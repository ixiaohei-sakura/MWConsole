## 程序介绍

MWConsole, mcdr-web-console.

是基于但不依赖mcdr而制作的方便使用的网页管理终端，即插即用.

后续程序完整后，将加入像mcdr一样的插件加载，并且会越来越完善.

后端: python + c++(一部分数据处理)

前端: js + css + html
